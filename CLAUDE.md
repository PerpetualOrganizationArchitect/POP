# CLAUDE.md

## Commands
- `forge build` - Build contracts (run first)
- `forge test` - Run tests
- `forge fmt` - Format code

## Rules
- Do NOT run `foundryup` - Foundry is pre-installed
- ALWAYS run `forge fmt` before commits and PRs

## Project
Solidity smart contracts for decentralized organizations (Foundry/Solidity ^0.8.20).
See `docs/` for documentation, `script/README.md` for deployment.
